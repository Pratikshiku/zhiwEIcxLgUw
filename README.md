### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录（数据加密）
2. 注册
3. 首页
4. 常用资料
5. 举报资料
6. 论坛交流
7. 公共文化
8. 村民举报
9. 个人中心
10. 我的发布
11. 我的收藏


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 常用资料管理
5. 举报处理
6. 论坛交流
7. 村民举报
8. 关于我们
9. 轮播图管理
10. 系统简介
11. 公共文化
